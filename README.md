# AI Chatbot using Python

## Description
This is a simple AI chatbot built using Python. It responds to user messages based on predefined intents.

## Features
- Responds to greetings
- Answers basic questions
- Easy to customize
- Beginner friendly

## Technologies Used
- Python
- JSON

## How to Run

Step 1: Clone the repository
git clone https://github.com/yourusername/chatbot-python.git

Step 2: Navigate to folder
cd chatbot-python

Step 3: Run the chatbot
python main.py

## Example

You: hello
Bot: Hi there!

You: who are you
Bot: I am a Python chatbot.
